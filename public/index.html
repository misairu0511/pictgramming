ピクトグラミング

Java の文でピクトグラムの全体や体の部位を回して動かす練習画面

表示エリア

体の部位にカーソルを合わせると、Java で使う名前が表示されます。

実行結果

Java 入力

クラスや main は不要です。一文ずつ書いて実行できます。

使える文
  • move(数字); 全体を前後に進める（マイナスで後ろ）
  • rotate(数字); 全体を回す（プラスで右、マイナスで左）
  • rotatePart("体の部位", 数字); 体の部位を回す（プラスで右、マイナスで左）
体の部位名
head body leftArm rightArm leftLeg rightLeg
