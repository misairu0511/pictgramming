ピクトグラミング

Java の文でピクトグラムの全体や体のパーツを動かす練習画面

表示エリア

右側に書いた Java 文の結果が、ゆっくりした動きで表示されます。

実行結果

Java 入力

クラスや main は不要です。一文ずつ書いて実行できます。

使える文
  • moveForward(80); 全体を前に進める
  • moveBack(40); 全体を後ろに下げる
  • turnLeft(90); 全体を左に回す
  • turnRight(90); 全体を右に回す
  • movePart("head", 0, -20); パーツを動かす
  • rotatePart("leftArm", -45); パーツを回す
  • moveHead(0, -20); 頭を動かす短い書き方
  • rotateLeftArm(-45); 左腕を回す短い書き方
  • resetParts(); パーツの位置を戻す
  • setColor("blue"); 色を変える
  • stamp(); 今の姿を残す
