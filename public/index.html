ピクトグラミング

Java の文でピクトグラムの全体や体の部位を回して動かす練習画面

表示エリア

体の部位にカーソルを合わせると、Java で使う名前が表示されます。

実行結果

Java 入力

クラスや main は不要です。一文ずつ書いて実行できます。

使える文
  • 移動(数字); 全体を前後に進める（マイナスで後ろ）
  • 回転(数字); 全体を回す（プラスで右、マイナスで左）
  • 部位回転("体の部位", 数字); 体の部位を回す（プラスで右、マイナスで左）
  • 掴む(); 近くの画像（ヒヨコ）を掴む
  • 離す(); 掴んでいる画像をその場に離す
