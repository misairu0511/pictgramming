# 🎭 ピクトグラミング

ピクトグラム（絵記号）を使ったビジュアルプログラミング環境です。
ブロックをドラッグ＆ドロップしてキャラクターを動かしましょう！

## 🚀 起動方法

```bash
npm install
npm start
```

ブラウザで `http://localhost:3000` を開いてください。

## 📦 ファイル構成

```
pictogramming/
├── server.js          # Node.js / Express サーバー
├── package.json
├── public/
│   ├── index.html     # メインUI
│   ├── css/
│   │   └── style.css  # スタイル
│   └── js/
│       ├── engine.js  # プログラム実行エンジン
│       └── app.js     # アプリケーション本体
└── README.md
```

## 🧩 使い方

1. 左のパレットからブロックをドラッグして中央のプログラムエリアに置く
2. ブロックのパラメータ（回数・距離・角度）を調整する
3. **▶ 実行** ボタンを押してキャラクターを動かす

## 🔧 ブロック一覧

| ブロック | 説明 |
|--------|------|
| 🔁 くりかえす | 以降のブロックをN回繰り返す |
| ⏳ まつ | N秒待つ |
| ⬆️ まえへすすむ | Npx前進 |
| ⬇️ うしろへさがる | Npx後退 |
| ↰ ひだりをむく | 左にN度回転 |
| ↱ みぎをむく | 右にN度回転 |
| 🔴🔵🟢 いろをかえる | キャラクターと軌跡の色を変える |
| 🖊️ スタンプ | 現在位置にスタンプを残す |
| 🔔 ピッ | 効果音を鳴らす |
| 🎉 かんせい | 完成音を鳴らす |
